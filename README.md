- Express;
- Typescript;
- ESLint;
- Prettier;
- Celebrate;
- Multer;
- CORS;
- JWT;
- BCrypt;
- Nodemailer;
- Handlebars;
- Postgres;
- Redis;
- TypeORM;
- Docker;
